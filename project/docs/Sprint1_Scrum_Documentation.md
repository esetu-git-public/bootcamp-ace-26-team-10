- Conducted Sprint Planning.

- Created and managed GitHub Issues.
- Monitored daily team progress.
- Coordinated communication between team members.
- Tracked completion of assigned tasks.
- Updated Scrum documentation.
- Ran daily standups and flagged blockers early.
- Sent progress updates to the Product Owner/Business Owner.
